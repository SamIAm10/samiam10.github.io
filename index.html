Scholarship Matching Website

Contract Project

As a developer/analyst, I was hired to work on a website that matches incoming college students with scholarships that they can realistically obtain, based on their individual merits, interests, hobbies, and other unique attributes. I learned that there are many students that don't even try to apply for financial aid, because they often feel overwhelmed by the immense number of scholarship opportunities and their strict requirements. This tool is designed to save time and effort by directly linking them to applications with higher chances of success. For those who can't afford the rising cost of tuition, this is an essential service for their transition into higher education.

My core functions in this project involved front and back-end web development (working closely with AWS, Linux, SSH, and SQL/NoSQL databases), code review, research, development of new and existing features, data analysis, tech consulting, and production of written reports with comprehensive system diagrams.

HTML CSS JavaScript jQuery Node.js AWS Linux SSH MongoDB SQL
